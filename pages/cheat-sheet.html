Objects are cool! this is how you initilize one:

var obj = {}

If you wanted to initilize one already with a few properties it might look like this:

var car = {make: "chevy", model: implala}

You could access any of those values by calling something like this:

console.log(car.make)

if you wanted to get really crazy you could even iterate through all the properties in an object.

for (var x in car) { console.log(x) }

If you wanted, you could git your object functions!

car.rev = function() {console.log("vroom vroom"}

One method i have found particularly useful is the has own property method, which returns a boolean based on wether or not an object contains a property.

car.hasOwnProperty(make) #=> true

Or maybe you want to merge a couple of objects. This could be done like this:

var boat = {} var train = {} var vehicles = {} Object.assign(vehicles, train, boat, car)
